## 正则表达式符号说明
1. L：大写字母；l：小写字母
2. d：除0以外的数字
3. C：所有的除双引号“”外的字母（用于字符串识别）

## 状态机转换说明
每读入一个字符，进行如下判断：
1. isdigit(): 判断是否是1-9的数字，如果是，则input为“d”
2. isUpperLetter():判断是否是大写字母，如果是，则input为“L"
3. isLowerLetter():判断是否是小写字母，如果是，则input为“l"
4. stringIdentifier():如果前一个字符是双引号“，说明接下来的输入可能是字符串，那么对于后面读入的每一个字符，只要他不是双引号”，就都将input设为“C”，直到读到双引号为止（表示字符串结束）。当然如果一直没有读到双引号，那就输出错误。
5. isKeyWord(): 如果识别出来是IDN，判断其是否在KW中。注意这里要对ORDER和GROUP做特别判断，如果当前是ORDER或GROUP的话，需要再往后读几个字符，看看是否是“ BY”，如果是，输出KW；否则，只认为他是IDN
6. isOP(): 如果是IDN，还需要判断他是不是OP（AND，OR，XOR），是则输出OP，否则输出IDN


## final state说明
s1: IDN

s2: INT

s3: INT(0)

s4: FLT

s5,s6,s7: OP

s8: OP, SE, STR
(所以如果到了s8的话，就需要判断前一个字符是什么，如果是双引号“，那就是STR；如果是(,)和,那就是SE；其余情况都是OP)
